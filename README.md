# multi-choice-system

A multiple choice exam system on browser.

# Install

* `bash propagate_env.sh`

### Frontend

* node ^14.15.0
* yarn ^1.22.x
* `yarn install`

### Backend

* python ^3.8.x
* can install directly with `pip install -r requirement.txt`
* or:

  ```
  pip install virtualenv
  virtualenv venv
  source venv/bin/activate
  pip install -r requirement.txt
  deactivate
  ```
